from mearm import Mearm

m = Mearm()

def wave_arm(times):
    for i in range(times):
        m.moveLowerTo(85)
        m.moveUpperTo(0)
        m.moveBaseTo(85)
        m.openGrip()
        m.closeGrip()
        m.moveBaseTo(-85)
        m.moveLowerTo(20)
        m.moveBaseTo(0)
        m.openGrip()
        m.closeGrip()
        m.moveLowerTo(85)
        m.moveUpperTo(0)

wave_arm(1)

Controlling MeArm with Javascript

Use these simple commands to get started controlling MeArm:

  • mearm.openGrip() - open the grip
  • mearm.closeGrip() - close the grip
  • mearm.moveBaseTo(angle) - turn the base to the angle
  • mearm.moveLowerTo(angle) - turn the lower part to the angle
  • mearm.moveUpperTo(angle) - turn the upper part to the angle
  • mearm.moveGripTo(angle) - turn the grip to the angle
